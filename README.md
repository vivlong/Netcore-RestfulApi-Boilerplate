# 概述

基于 ServiceStack, .net core 2 的脚手架项目。

# 安装

## 安装 .net core sdk
首先到 [.net core](https://www.microsoft.com/net/download/all) 页面下载最新的 2.2 版本的 sdk。
VS2019 NuGet Resore all packages. 

## 环境
  * ServiceStack.net &nbsp;&nbsp;&nbsp;&nbsp;(/API;/API.ServiceInterface;/API.ServiceModel)  
  * dotNet Core 2.2
